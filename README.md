# StockLens · 智能股票分析

> 🇨🇳 A股（DeepSeek）+ 🇺🇸 美股（GPT-4o-mini）双市场 AI 分析工具

实时爬取行情数据，结合 AI 对自选股、持仓、市场机会做全面分析，一个页面同时管理 A 股和美股。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install flask yfinance openai requests
```

### 2. 启动

```bash
python app.py
```

浏览器打开 [http://localhost:5000](http://localhost:5000)，在右上角「⚙ 设置」中填入 API Key，即可开始分析。

### 3. API Key 获取

| Key | 用途 | 获取地址 |
|---|---|---|
| DeepSeek API Key | A股分析 + 政策主线 + A股聊天 | [platform.deepseek.com](https://platform.deepseek.com) |
| OpenAI API Key | 美股分析 + 美股诊股 + 美股聊天 | [platform.openai.com](https://platform.openai.com) |

## ✨ 功能概览

### 🇨🇳 A股（DeepSeek）

#### 自选股分析
- 实时抓取涨幅榜、板块资金流向、连板股、大盘指数
- 每只股票显示：当前价、涨跌幅、MA 均线、K线走势（可切换5日/30日/1年/5年）
- AI 给出：**趋势判断 / 板块热度 / 入场条件 / 止损信号**

**评分权重：** 资金流向 40% · 板块热度 30% · 量价结构 20% · 均线趋势 10%

#### 持仓诊断
- 自动计算成本、现值、浮盈浮亏
- AI 一键诊断所有持仓，逐一给出 持有 / 加仓 / 减仓 / 止损 建议

**止损优先级（从高到低）：**
1. 放量下跌（主力出货）
2. 板块退潮（龙头炸板、涨停数减少）
3. 题材逻辑被证伪
4. 浮亏 >10% 且以上信号同时出现

#### AI 荐股（防幻觉机制）
- 从东方财富**主力净流入 Top30** 建立候选池，AI 只能从中选股
- 候选池为空 / 所有标的涨幅 >9% → 返回空列表，宁可不推
- **推荐数量：5~8 只**（短线 3~4 只 + 中长线 2~4 只）

**推荐逻辑优先级：**
1. 实时资金（板块净流入 + 涨幅榜集中度）
2. 市场情绪（连板高度 + 情绪冷热）
3. 政策共振（加分项，不单独构成推荐理由）

#### 🏛 政策主线（七大方向）
对七大中长期政策板块做独立深度分析：

| 板块 | 代表方向 |
|---|---|
| AI算力/大模型 | CPO / 光模块 / 液冷服务器 |
| 半导体/芯片 | 国产替代 / 存储 / 设备 / 材料 |
| 人形机器人 | 电机 / 减速器 / 传感器 |
| 低空经济 | eVTOL / 无人机 / 空管 |
| 国防军工 | 信息化装备 / 航空发动机 / 导弹 |
| 新能源/储能 | 大储 / 工商储 / 出海 |
| 创新药/医疗器械 | 国产创新药出海 + 集采后修复 |

每个板块判断所处周期（酝酿期 → 启动期 → 加速期 → 调整期 → 衰退期），给出对应仓位建议和中长线个股筛选。

### 🇺🇸 美股（GPT-4o-mini）

#### 自选股分析
六步固定框架：

1. **财报风险**（最重要）— 4周内有财报？浮盈 >20% 建议财报前减仓 30~50%
2. **板块ETF动能（40%）** — 对应ETF是否跑赢SPY？
3. **相对强度（20%）** — 个股近一月 vs SPY
4. **分析师情绪（20%）** — 近期升评 / 降评 / 调价目标
5. **宏观敏感度（10%）** — 利率 / 美元 / VIX 影响
6. **技术（10%）** — MA50/MA200 只看趋势方向，不作买卖信号

#### 持仓诊断
五步框架：

1. 判断股票类型（成长 / 动能 / 价值 / 主题）
2. 财报日历检查 — 核心决策依据
3. 板块ETF健康度
4. VIX 仓位管理（>25 减仓 / >30 清仓）
5. 盈亏纪律（浮盈 >25% 必须给减仓方案；亏损 >15% 建议止损）

#### AI 荐股（防幻觉机制）
从 Yahoo Finance **涨幅榜 + 成交量榜**建立候选池（最多40只），AI 只能从中选股。候选池不足时自动补充 S&P 500 核心成分股，确保始终有推荐输出。

**推荐分三类：**

| 类型 | 数量 | 逻辑 |
|---|---|---|
| ⚡ 短线动能 | 1~2 只 | 今日ETF领涨 + 放量突破，ETF反转立刻离场 |
| 📅 财报催化 | 2~3 只 | 有业绩超预期历史的即将财报股 |
| 📈 中长线主线 | 2~3 只 | AI/医疗/能源，ETF资金流入 + 分析师上调共振 |

## 🧠 A股 vs 美股 核心差异

| | 🇨🇳 A股 | 🇺🇸 美股 |
|---|---|---|
| 最大风险 | 板块情绪退潮、游资撤退 | 财报暴雷 |
| 核心节奏 | 题材热度周期（天级别） | 财报周期（季度级别） |
| 止损第一优先 | 放量下跌 / 板块退潮 | 财报 EPS 低于预期 |
| 推荐候选池来源 | 东方财富主力净流入 Top30 | Yahoo 涨幅榜 + 成交量榜 Top40 |
| AI 模型 | DeepSeek `deepseek-chat` | GPT-4o-mini |
| 单次分析费用 | ~$0.05~0.10 | ~$0.01~0.03 |

## 📌 注意事项

- 分析卡住不动 → 点右上角「↺ 重置」后重试，或删除 `data/china/.status.json` / `data/us/.status.json` 后重启
- 历史结果自动保存在 `archive/`，保留最近 14 天 / 最多 50 份
- 政策主线分析约需 60~90 秒，独立触发
- **本工具仅供参考，不构成投资建议，投资有风险，入市须谨慎**
